How to Use

  • 1. 회원가입 및 로그인을 진행합니다.
  • 2. 가명화 처리가 필요한 파일을 서버에 업로드합니다.
  • 3. 파일 내용을 확인하고 가명화할 정보를 선택합니다.
  • 4. 추가로 가명화할 정보를 직접 입력합니다.
  • 5. Finish 버튼을 눌러 가명화 처리를 완료합니다.
  • 6. GPT 버튼을 눌러 업로드한 파일에 대해 질문을 할 수 있습니다.
  • 7. History 페이지에서 업로드한 파일 내역을 확인할 수 있습니다.
Tip: 가명정보 처리 결과를 증명하는 증명서를 발급받을 수 있습니다.
